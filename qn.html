$ innerText returns the text content of the element
$ textContent returns the text content of the element

What is event delegation in the DOM?

Event delegation is a technique in JavaScript that allows you to attach single event listeners to a parent element instead of attaching multiple event listeners to each child element.

What is event bubbling in the DOM?

Event bubbling is a technique in JavaScript that allows event to propagate through the DOM.
